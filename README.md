# Standing Ovation Model

A Mesa implementation of Miller and Page's Standing Ovation Problem (2004).

## Files

- `agents.py` — AudienceAgent class
- `model.py` — StandingOvationModel class
- `app.py` — Solara visualization

## Parameters

| Parameter | Options | Default |
|---|---|---|
| Threshold | 0.0 – 1.0 | 0.5 |
| Neighborhood | Five Neighbors, Cones | Five Neighbors |
| Update Mode | Synchronous, Asynchronous-Random, Asynchronous-Incentive-Based | Synchronous |

## Reference

Miller, J. H., & Page, S. E. (2004). The standing ovation problem. *Complexity*, 9(5), 8–16.
